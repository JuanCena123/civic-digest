# CivicDigest
App that summarizes different civic actions to promote an informed populous and transparent government

## Tech Stack
- Next.js (App Router)
- Supabase (Postgres + RLS)
- GitHub Actions (data ingestion)
- Vercel (hosting)
- OpenAI / Azure OpenAI (summarization)

## Features
- Automated ingestion of federal bills
- Searchable legislation database
- AI-generated TL;DRs
- Designed for future ML extensions
